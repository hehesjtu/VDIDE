# VDIDE: Vortex Dynamics-inspired Single-Image Detail Enhancement
This is the source code of paper named "Vortex Dynamics-inspired Single-Image Detail Enhancement". This paper is submitted to IEEE Knowledge-Based Systems.

[![MATLAB](https://img.shields.io/badge/MATLAB-R2024b+-blue.svg)](https://www.mathworks.com/products/matlab.html)

This repository provides the official MATLAB implementation of the **VDIDE** algorithm, as described in the research paper:

> **"Vortex Dynamics Inspired for Single-Image Detail Enhancement"**
> *Authors: [He Jiang], [Gang Yang], [Jiahe Jiang], et al.*

---

## 1. Introduction

**Abstract:** Traditional residual learning-based image detail enhancement algorithms often utilize greedy search mechanisms, which are susceptible to trapping in local optima. This limitation typically leads to detail degradation and a reduction in structural consistency. To address these issues, this study proposes a **Vortex Dynamics-inspired Image Detail Enhancement (VDIDE)** algorithm. 

The method models the image block matching process as the dynamical evolution of fluid particles within a two-dimensional residual space:

* **Global Exploration (Large-scale Advection):** The algorithm simulates advective transport driven by large-scale background flow fields to perform global exploration. By leveraging wide-area migration, it effectively circumvents local optimum traps common in greedy search mechanisms.
* **Local Exploitation (Centripetal Vortex Capture):** It leverages centripetal vortex patterns within the core region to achieve precise refinement of local features. Particles approach the global optimal solution along helical trajectories, balancing search breadth with convergence precision.
* **Composite Fitness Evaluation:** A composite fitness function integrates pixel differences, gradient edges, and texture features to drive the molecules toward a steady-state locking.

## 2. System Requirements

The code has been developed and verified on the following environment:

* **MATLAB Version**: R2021b or later (Recommended: R2025a)
* **Operating System**: Windows 10/11, macOS, or Linux

### Required Toolboxes:
* **Image Processing Toolbox**: Essential for image I/O, resizing, and gradient calculations.
* **Parallel Computing Toolbox**: Used for the `parfor` acceleration to optimize the heuristic search process.

## 4. Usage

We provide a user-friendly Graphical User Interface (GUI) to facilitate testing and parameter adjustment.

### Steps:
1.  Open MATLAB and navigate to the project directory.
2.  In the Command Window, run the interface by typing:
    ```matlab
    UI
    ```
3.  **Load Image**: Import your target image (supports `.png`, `.jpg`, `.bmp`, `.tif`).
4.  **Enhancement Factor**: Adjust the slider to control the intensity of detail enhancement (Default: 4.0).
5.  **Run**: Click the process button to execute the search-based enhancement.
6.  **Save**: Export the high-quality result to your local drive.

## 5. File Description

* `UI.m`: The main entry point script containing the GUI implementation and interaction logic.
* `VDIDE.p`: The pre-compiled binary file of the core VDIDE algorithm. 
  *Note: The core heuristic search algorithm is provided as a P-code file for copyright protection. It is functionally identical to the source code described in the paper.*
* `images/`: (Optional) Contains sample images for testing purposes.
* `results/`: Stores the corresponding enhanced images processed by the VDIDE algorithm, serving as a quality reference.
